Primary interactive button — use for the main action in any view (Save, Add Lead, Send Quote).

```jsx
<Button variant="primary" size="md" leadingIcon={<PlusIcon/>}>Add Lead</Button>
```

Variants: `primary` (brand blue, default CTA), `secondary` (white/outline), `ghost` (text only), `success` (green — confirm/book), `danger` (destructive). Sizes `sm | md | lg`. Use `fullWidth` in forms and mobile sheets. One primary per view.
