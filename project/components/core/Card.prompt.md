Surface container for grouped content — dashboard panels, list sections, profile blocks.

```jsx
<Card padding="md" accent>
  <CardHeader title="Upcoming Jobs" subtitle="Next 7 days" action={<Button variant="ghost" size="sm">View all</Button>} />
  …
</Card>
```

`padding`: none|sm|md|lg. `interactive` for clickable cards (hover lift). `accent` adds a brand-blue top rule.
