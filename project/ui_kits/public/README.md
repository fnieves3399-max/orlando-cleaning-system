# Public Site UI Kit — Orlando Cleaning Growth System

The **customer-facing** local landing page generated by the platform's Landing Page Builder. Open `index.html`. Self-contained (uses design tokens directly from `styles.css`); no bundle required.

## Page: "House Cleaning in Orlando, FL"
A complete local-service landing page with the full section set the builder produces:
- Sticky header (logo, nav, phone, **Get a quote**)
- **Hero** with headline, trust chips, and an **embedded quote form** (name, phone, address, cleaning type, frequency, beds/baths)
- Offer strip (navy) — "20% off first recurring clean"
- **Services** grid (Standard, Deep, Move-out, Airbnb turnover, Post-construction + helper card)
- **Benefits** (vetted/insured, flat pricing, 60-second booking, guarantee)
- **Reviews** (4.9★ social proof)
- **FAQ** (native `<details>` accordions)
- **CTA** band + **local SEO paragraph** (Orlando + neighborhoods)
- Navy footer (services, service areas, contact)

## Notes
- Demonstrates the **marketing/landing voice**: benefit-led, locally specific, warm. This is the one surface where a soft brand wash and a hero CTA gradient are allowed.
- Swap the SEO title/description, H1, neighborhood names, and offer to spin up the other landing-page types (Winter Park, Lake Nona, Kissimmee, Windermere, Clermont).
- Icons are inline Lucide-style SVGs matching the system's iconography.
